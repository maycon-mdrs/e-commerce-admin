# 🎲 Rodando o código

### FrontEnd
```bash
# Clone este repositório
$ git clone <https://github.com/maycon-mdrs/login-react>

# Acesse a pasta do projeto
$ cd login-react

# Instale as dependências
$ npm install

# Execute a aplicação em modo de desenvolvimento
$ npm start

# Acesse <http://localhost:3000>
```
